# Portafolio de Análisis de Datos

Este repositorio reúne **proyectos prácticos de análisis de datos** desarrollados con el objetivo de demostrar habilidades en **captura, limpieza, validación, análisis y visualización de información**, aplicadas a escenarios cercanos a contextos reales de negocio.

Los proyectos aquí presentados reflejan mi formación como **Ingeniero en Mecatrónica** con certificación en **Análisis de Datos por IBM**, así como mi interés en iniciar mi carrera profesional en áreas relacionadas con datos, análisis y soporte administrativo basado en información.

---

## Objetivo del portafolio

Mostrar de forma clara y estructurada la aplicación práctica de herramientas de análisis de datos para:

- Organizar y validar información  
- Analizar datos para obtener conclusiones útiles  
- Elaborar reportes y dashboards  
- Apoyar la toma de decisiones y el control de procesos  

---

## Proyectos incluidos

### Proyecto final – Análisis de habilidades tecnológicas en demanda

**Descripción:**  
Análisis exploratorio de datos enfocado en identificar **tendencias del mercado tecnológico**, utilizando información proveniente de encuestas de desarrolladores y fuentes públicas.

**Actividades realizadas:**
- Integración de datos desde múltiples fuentes  
- Limpieza y preparación de la información  
- Análisis exploratorio de datos  
- Identificación de tendencias y conclusiones orientadas a negocio  
- Visualización de resultados en dashboards  

**Herramientas:**  
Python (Pandas, Matplotlib), SQL, Excel, IBM Cognos Analytics

---

### Proyecto personal – Análisis de ventas (tienda ficticia)

**Descripción:**  
Proyecto enfocado en el análisis de ventas para una tienda simulada, con el objetivo de medir desempeño y comportamiento comercial.

**Actividades realizadas:**
- Definición de indicadores clave (KPIs)  
- Limpieza y organización de datos  
- Análisis de resultados  
- Creación de dashboard interactivo  

**Herramientas:**  
Excel, Power BI

---

## Herramientas y tecnologías

- **Excel:** limpieza de datos, tablas dinámicas, reportes  
- **SQL:** consultas, agregaciones, análisis de información  
- **Python:** análisis exploratorio de datos (Pandas, Matplotlib)  
- **Power BI:** dashboards interactivos  
- **IBM Cognos Analytics:** visualización y análisis

## Habilidades demostradas
